解构赋值

ES6 允许按照一定模式，从数组和对象中提取值，对变量进行赋值，这被称为解构（Destructuring）；
解构赋值是一种快速为变量赋值的简洁语法，本质上仍然是为变量赋值；
ES5 为变量赋值，只能直接指定值；
let a = 1
let b = 2
let c = 3

1. 数组的解构赋值

1.1 数组解构赋值概念

从数组中提取值，按照对应的位置，对变量赋值

数组的解构赋值可以用于交换变量

tips：要加分隔符号的情况，以以下情况开头：

() 立即执行函数；
[] 数组解构；

1.2 数组结构的细节

  1. 变量多，数组元素少
  1. 变量少，数组元素多
  1. 按照对应位置，依次对变量赋值
  1. 解构赋值允许指定默认值
  1. 如果等号右边不是数组（不可遍历的结构），那么会报错

1.3 多维数组解构

多位数组的结构，按照相应的结构写好，一一对应；

2. 对象的解构赋值

对象的解构，是按照属性名结构，可以交换位置。要求变量名必须和属性名一致才可以结构；

如：const { age, uname } = { uname: '周杰伦', age: 18 }

2.1 对象结构重命名

对象的解构赋值，可以重新命名；

语法： 旧变量名: 新变量名

结论：
对象解构的内部机制，是先找到同名属性；
然后再赋值给对应的变量，真正被赋值的是冒号后面的那个；
