对象的方法相关

1. 基本包装类型

按照常理来说，基本数据类型没有属性和方法；

基本包装类型：在JS底层，吧简单数据类型包装成了复杂数据类型（对象），这个时候，基本数据类型就可以使用相应的一些属性和方法；

2. Object 方法相关

现有对象

2.1 Object.keys()

Object.keys() 会返回对象的所有属性名组成的字符串数组；

2.2 Object.values()

Object.values() 会返回对象的所有属性值组成的数组

2.3 Object.assign()

Object.assign() 对象的拷贝 浅拷贝

target 目标对象

sources 源对象， 一个或者多个， 把他的属性复制给目标对象

返回值： 得到复制属性后的目标对象

3. Array方法相关

3.1 arr.reduce()

arr.reduce() ,该方法可以对数组的每个元素，都执行一个自定义的reduce函数，将其结果汇总为单个返回值；

语法：arr.reduce(function(){}, initValue)

参数：callbackFn 回调函数 必选 √；

​ initValue 初始值 （可选）；

callbackFn 回调函数里的参数：

previousValue ： 上一次调用callbackFn的返回值，必选 √

currentValue ： 当前元素 必选 √

currentIndex ： 当前元素的索引（可选）

array ： 源数组 （可选）

注意：第一次调用的时候，如果有初始值，则 prev 为初始值，否则 prev 为数组的第一个元素arr[0]

3.2 arr.find()

语法：arr.find(cbFn)

作用：返回数组中满足提供的测试函数的第一个元素的值。否则返回 undefined；

参数： cbFn 回调函数；

cbFn 的参数

第一个参数：item 当前的元素；

第二个参数：index 当前元素的索引号(可选)；

返回值：数组中第一个满足所提供测试函数的元素的值，否则返回 undefined；

3.3 arr.findIndex()

语法：arr.find(cbFn);

作用：返回数组中满足提供的测试函数的第一个元素的索引。否则返回 -1;

参数： cbFn 回调函数

cbFn 的参数:

第一个参数：item 当前的元素;

第二个参数：index 当前元素的索引号(可选);

返回值：返回数组中第一个满足所提供测试函数的元素的索引，否则返回 -1;

3.4 arr.every()

arr.every()

作用:检测数组中所有的元素是否都符合指定的条件,如果都符合,返回 true;

参数： cbFn 回调函数;

cbFn 的参数:

第一个参数：item 当前的元素;

第二个参数：index 当前元素的索引号(可选);

返回值:Boolean 所有的元素都通过,true ,否则 false;

注意:因为这个every方法有返回值,所有cbFn里面需要有 return;

3.7 Array.from()

Array.from() 静态方法 把伪数组,转换为真数组;

3.8 Array.filter()

Array.filter() filter 筛选,过滤的意思;创建一个新的数组, 新数组的元素是 通过筛选后的元素;

filter() 接收一个函数作为参数, 函数返回true或者false;

true: 表示通过测试, 保留当前元素, false, 不保留;

函数的第一个参数 当前处理的元素 item 必须写;

函数的第二个参数 当前处理的元素的索引号 index;

返回值: 一个新的数组, 由通过测试的元素组成;

4. String 方法相关

4.1 str.split()

使用指定分隔符,将字符串分割,得到一个字符串数组；

4.2 str.sunstring()

语法: str.sunstring(开始的索引号[, 结束的索引号])

作用:字符串截取;

注意:

  • 如果省略indexEnd(结束的索引号),一直取到最后;
  • 前闭后开区间[start, end), 结束索引号不包含咋子截取的字符串内;

4.3 str.startsWith()

语法: str.startsWith(searchString[, position]);

作用: 判断是否以某个字符串开头;

参数:

第一个参数:要搜索的字符串;

第二个参数:在str中搜索的开始位置,默认为0(可选);

返回值: true/false;
